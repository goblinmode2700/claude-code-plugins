# Claude Code Plugins by NecessitasLabs

Production-grade plugins for [Claude Code](https://claude.ai/claude-code) — the AI coding agent by Anthropic.

## Plugins

### claude-md-architect
Generate a structured CLAUDE.md governance file for your project. Produces session modes, code standards, hard constraints, and architecture rules tailored to your tech stack.

**Usage:** Install the plugin, then run `/claude-md-architect` in Claude Code.

### sprint-manager
Structure AI-assisted development sprints with phase gates, quality checkpoints, and progress tracking. Prevents scope creep and keeps Claude Code focused.

**Usage:** Install the plugin, then run `/sprint-manager` in Claude Code.

## Installation

```
/plugin marketplace add goblinmode2700/claude-code-plugins
```

Or install individual plugins by cloning this repo and adding the plugin directory to your Claude Code config.
